# AI NPC Dialogue System

## Overview

This project is an AI-powered NPC dialogue system developed in Unity.
It allows players to interact with NPCs in a first-person environment using dynamic, choice-based conversations.

## Features

* First-person player interaction (FPS)
* AI-driven NPC dialogue generation
* Dynamic dialogue choices
* Conversation memory system (planned)
* Modular and scalable architecture

## Technologies

* Unity 6 (URP)
* C#
* REST API (LLM integration)
* JSON-based dialogue system

## Project Structure

```
Assets/
│── Scenes/
│── Scripts/
│   ├── Player/
│   ├── NPC/
│   ├── Dialogue/
│   ├── AI/
│── Prefabs/
│── UI/
```

## How It Works

1. Player approaches an NPC
2. Presses interaction key (E)
3. Dialogue UI appears
4. AI generates NPC response and choices
5. Player selects a response
6. Conversation continues dynamically
