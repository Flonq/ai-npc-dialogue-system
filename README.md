## вњЁ Overview

An AI-powered NPC dialogue system built in Unity.  
The player explores in **first-person**, looks at an NPC, presses **E**, and starts a fully **dynamic conversation**.  
Each turn the NPC reply and the **3 player choices** are generated by a **local LLM** running through **LM Studio**, returned in a strict JSON schema.

## рџ› пёЏ Setup

### 1) Unity
1. Open the project in **Unity 6 (URP)**.
2. Install **Newtonsoft.Json** via Package Manager  
   (Add package by name в†’ `com.unity.nuget.newtonsoft-json`).
3. Open `Scenes/SampleScene.unity`.

### 2) LM Studio
1. Install [LM Studio](https://lmstudio.ai).
2. Download a small instruct model (e.g. `qwen2.5-0.5b-instruct`).
3. In **Developer / Local Server**, load the model and click **Start Server**.  
   Default endpoint: `http://localhost:1234`.

### 3) DialogueManager (Inspector)
| Field | Suggested value |
|-------|------------------|
| Endpoint | `http://localhost:1234/v1/chat/completions` |
| Model | `qwen2.5-0.5b-instruct` |
| Temperature | `0.4` |
| Max Tokens | `200` |

Press **Play**, walk to the NPC, press **E**, talk away.
